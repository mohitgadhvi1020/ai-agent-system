# AI Agent System

A multi-step AI agent built on a **ReAct-style reasoning loop** — observe →
think → act → repeat. Six tools, dual Claude/GPT support, and a zero-key
heuristic demo mode so it runs live with no API keys.

- **UI:** the chat + live reasoning trace is served at `/`
- **API docs:** Swagger at `/docs`

This Space runs the FastAPI app from its Dockerfile. Source:
<https://github.com/mohitgadhvi1020/ai-agent-system>
